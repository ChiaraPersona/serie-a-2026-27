# Dati grezzi

## Statistiche arbitrali

Le risposte complete del provider sono conservate in `referee-stats/<provider>/<stagione>/<competizione>/<fixture-id>.json`, con URL e data di recupero. Le credenziali vengono inviate soltanto nell'header HTTP e non devono essere scritte negli snapshot.

Conservare qui gli snapshot delle fonti senza modificarli, insieme a data di acquisizione e URL. Nessun dato ufficiale è stato importato nella fase demo.
